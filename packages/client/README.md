# @bucketmate/client

S3 client package for BucketMate.

## Installation

```bash
npm install @bucketmate/client
```

## Usage

```typescript
import { createS3Client } from '@bucketmate/client';

// Create an S3 client
createS3Client();
```

## Development

```bash
# Build the package
pnpm run build

# Watch for changes
pnpm run dev

# Clean build artifacts
pnpm run clean
```
